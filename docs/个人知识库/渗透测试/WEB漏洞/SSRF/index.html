дҪңиҖ…дҝ®и®ўж—¶й—ҙ

жјҸжҙһд»Ӣз»ҚвҖӢ

SSRFпјҡе…Ёз§° Server-Side Request ForgeryпјҢеҚі жңҚеҠЎеҷЁз«ҜиҜ·жұӮдјӘйҖ гҖӮжҳҜдёҖдёӘз”ұж”»еҮ»иҖ…жһ„йҖ иҜ·жұӮпјҢеңЁзӣ®ж ҮжңҚеҠЎз«Ҝжү§иЎҢзҡ„дёҖдёӘе®үе…ЁжјҸжҙһгҖӮж”»еҮ»иҖ…еҸҜд»ҘеҲ©з”ЁиҜҘжјҸжҙһдҪҝжңҚеҠЎеҷЁз«Ҝеҗ‘ж”»еҮ»иҖ…жһ„йҖ зҡ„д»»ж„ҸеҹҹеҸ‘еҮәиҜ·жұӮпјҢзӣ®ж ҮйҖҡеёёжҳҜд»ҺеӨ–зҪ‘ж— жі•и®ҝй—®зҡ„еҶ…йғЁзі»з»ҹгҖӮз®ҖеҚ•жқҘиҜҙе°ұжҳҜеҲ©з”ЁжңҚеҠЎеҷЁжјҸжҙһд»ҘжңҚеҠЎеҷЁзҡ„иә«д»ҪеҸ‘йҖҒдёҖжқЎжһ„йҖ еҘҪзҡ„иҜ·жұӮз»ҷжңҚеҠЎеҷЁжүҖеңЁеҶ…зҪ‘иҝӣиЎҢж”»еҮ»гҖӮ

жјҸжҙһеҺҹзҗҶвҖӢ

SSRFе…Ёз§°дёәServer-side Request Fogery,дёӯж–Үеҗ«д№үдёәжңҚеҠЎеҷЁз«ҜиҜ·жұӮдјӘйҖ пјҢжјҸжҙһдә§з”ҹзҡ„еҺҹеӣ жҳҜжңҚеҠЎз«ҜжҸҗдҫӣдәҶиғҪеӨҹд»Һе…¶д»–жңҚеҠЎеҷЁеә”з”ЁиҺ·еҸ–ж•°жҚ®зҡ„еҠҹиғҪпјҢжҜ”еҰӮд»ҺжҢҮе®ҡзҡ„URLең°еқҖиҺ·еҸ–зҪ‘йЎөеҶ…е®№пјҢеҠ иҪҪжҢҮе®ҡең°еқҖзҡ„еӣҫзүҮгҖҒж•°жҚ®гҖҒдёӢиҪҪзӯүзӯүгҖӮжјҸжҙһURLзӨәдҫӢ: http://xxx.com/api/readFiles?url=http://10.1.11/xxx

жјҸжҙһеҲ©з”ЁвҖӢ

еҶ…зҪ‘и®ҝй—®вҖӢ

дҪҝз”ЁhttpеҚҸи®®еҜ№еҶ…зҪ‘зҡ„Webеә”з”ЁиҝӣиЎҢи®ҝй—®

?url=http://127.0.0.1/flag.php

php://output жҳҜдёҖдёӘеҸӘеҶҷзҡ„ж•°жҚ®жөҒпјҢ е…Ғи®ёдҪ д»Ҙ print е’Ң echo дёҖж ·зҡ„ж–№ејҸ еҶҷе…ҘеҲ°иҫ“еҮәзј“еҶІеҢәгҖӮ

php://memory е’Ң php://temp жҳҜдёҖдёӘзұ»дјјж–Үд»¶ еҢ…иЈ…еҷЁзҡ„ж•°жҚ®жөҒпјҢе…Ғи®ёиҜ»еҶҷдёҙж—¶ж•°жҚ®гҖӮ дёӨиҖ…зҡ„е”ҜдёҖеҢәеҲ«жҳҜ php://memory жҖ»жҳҜжҠҠж•°жҚ®еӮЁеӯҳеңЁеҶ…еӯҳдёӯпјҢ иҖҢ php://temp дјҡеңЁеҶ…еӯҳйҮҸиҫҫеҲ°йў„е®ҡд№үзҡ„йҷҗеҲ¶еҗҺпјҲй»ҳи®ӨжҳҜ 2MBпјүеӯҳе…Ҙдёҙж—¶ж–Үд»¶дёӯгҖӮ дёҙж—¶ж–Үд»¶дҪҚзҪ®зҡ„еҶіе®ҡе’Ң sys_get_temp_dir() зҡ„ж–№ејҸдёҖиҮҙгҖӮ php://temp зҡ„еҶ…еӯҳйҷҗеҲ¶еҸҜйҖҡиҝҮж·»еҠ  /maxmemory:NN жқҘжҺ§еҲ¶пјҢNN жҳҜд»Ҙеӯ—иҠӮдёәеҚ•дҪҚгҖҒдҝқз•ҷеңЁеҶ…еӯҳзҡ„жңҖеӨ§ж•°жҚ®йҮҸпјҢи¶…иҝҮеҲҷдҪҝз”Ёдёҙж—¶ж–Үд»¶гҖӮ
