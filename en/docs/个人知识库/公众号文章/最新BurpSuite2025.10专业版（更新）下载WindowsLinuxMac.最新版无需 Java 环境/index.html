最新BurpSuite2025.10专业版（更新）下载Windows/Linux/Mac.最新版无需 Java 环境

WINDOWS安装教程​

首先我们下载对应系统的安装包，例如我是 windows amd64 架构的系统，我选择下载 burpsuite_pro_windows-x64_v2025_10_1.exe 、 burpsuitloader-4.11.22-all.jar、burpsuite hash.txt

计算其 安装包的md5是否与官网提供的md5一致


PS C:\Users\Administrator\Downloads> Get-FileHash -Path "burpsuite_pro_windows-x64_v2025_10_1.exe" -Algorithm MD5

Algorithm       Hash                                                                   Path
---------       ----                                                                   ----
MD5             646508514E8FBB19224BC7F118840931                                       C:\Users\Administrator\Downlo...


PS C:\Users\Administrator\Downloads>

若 md5 值匹配了就选择双击安装包，并选择安装位置。

安装完成后，将burpsuitloader-4.11.22-all.jar 移动到 Burp 的安装目录

然后，将下面的文件替换成下方的内容

C:\Program Files\BurpSuitePro\BurpSuitePro.vmoptions
-XX:MaxRAMPercentage=50
-include-options settings.vmoptions
-include-options user.vmoptions
--add-opens=java.desktop/javax.swing=ALL-UNNAMED
--add-opens=java.base/java.lang=ALL-UNNAMED
--add-opens=java.base/jdk.internal.org.objectweb.asm=ALL-UNNAMED
--add-opens=java.base/jdk.internal.org.objectweb.asm.tree=ALL-UNNAMED
--add-opens=java.base/jdk.internal.org.objectweb.asm.Opcodes=ALL-UNNAMED
-noverify
-javaagent:burpsuitloader-4.11.22-all.jar=loader,han

然后找到 burp 安装包安装的 java.exe,它位于目录下的这个文件中,启动注册机

& "C:\Program Files\BurpSuitePro\jre\bin\java.exe" -jar .\burpsuitloader-4.11.22-all.jar

MACOS 安装教程​

首先我们下载对应系统的安装包，例如我是 macos m2 架构的系统，我选择下载 burpsuite_pro_macos_arm64_v2025_10_1.dmg 、 burpsuitloader-4.11.22-all.jar、burpsuite hash.txt

计算其 安装包的md5是否与官网提供的md5一致

➜ ~ md5sum burpsuite_pro_macos_arm64_v2025_10_1.dmg
4ddf2e8f7ca2c11a3f9dc46e75e594db  burpsuite_pro_macos_arm64_v2025_10_1.dmg

若你的安装包的 md5 与官网提供的不一致，那么请不要安装

双击安装包，将 Burp Suite Professional.app 移动到 Applications

然后将burpsuitloader-4.11.22-all.jar 移动到 /Applications/Burp Suite Professional.app/Contents/Resources/app目录下

cp burpsuitloader-4.11.22-all.jar '/Applications/Burp Suite Professional.app/Contents/Resources/app'

然后，将下面的文件替换成下方的内容

/Applications/Burp Suite Professional.app/Contents/vmoptions.txt
-XX:MaxRAMPercentage=50
-include-options settings.vmoptions
-include-options user.vmoptions
--add-opens=java.desktop/javax.swing=ALL-UNNAMED
--add-opens=java.base/java.lang=ALL-UNNAMED
--add-opens=java.base/jdk.internal.org.objectweb.asm=ALL-UNNAMED
--add-opens=java.base/jdk.internal.org.objectweb.asm.tree=ALL-UNNAMED
--add-opens=java.base/jdk.internal.org.objectweb.asm.Opcodes=ALL-UNNAMED
-noverify
-javaagent:burpsuitloader-4.11.22-all.jar=loader,han

然后打开注册机，运行一下命令

cd "/Applications/Burp Suite Professional.app/Contents/Resources/app" && "/Applications/Burp Suite Professional.app/Contents/Resources/jre.bundle/Contents/Home/bin/java" -jar burpsuitloader-4.11.22-all.jar

如果出现以下错误

运行

sudo xattr -r -d com.apple.quarantine /Applications/Burp\ Suite\ Professional.app

总结​

虽然安装步骤些许繁琐，好在不依赖任何环境，你可以移除任何Java 环境，并且后续更新，双击 Burp 最新安装包，即可更新，无需重复激活。那么回复【BurpSuite2025.10】即可领取上述的所有安装环境教程、以及所用到的 Burp 插件
