04.基于vless的科学上网-自动化

基于vless的科学上网 - 自动化

前沿

这一篇文章是根据 基于vless的科学上网 的 拓展 ，如果你没看过也没关系 看这一篇也足够了

准备

  • (ubuntu、centos都可以) vps、域名 （如果有多台vps的话，更换域名后，运行即可多台的节点都可以添加上）
  • 一个github仓库（公开的，非公开的库访问困难，根据github的API可以访问私库，不过得配置http请求头，所以不支持的客户端是访问不到的），一个github的 token

创建github仓库

建立仓库，点击new

填好名字，并点击 Create repository

创建成功自动跳转，代表你成功了

创建token

setting -> Developer settings

创建经典的token 不会过期

填好，创建就行了

命令

# 建立主目录,并进入目录
mkdir /proxy && cd /proxy
# 建立配置文件, 并导入证书 域名.key、域名.crt 至 ./conf/ssl/
# 复制域名证书内容至文件中
mkdir ./conf/ssl -p

导入脚本文件,填写相关参数 start.py -> start.py

运行，根据 提示排错 第一次安装会有点慢

python3 start.py

运行成功的画面

然后导入订阅链接即可

不用了，删除

# 进入到目录中
cd /proxy
# 停掉docker容器
docker-compose down
# 删除整个目录
rm /proxy -rf

客户端选择

  • windows：winxray、v2rayN
  • linux：v2rayA
  • ios：Shadowrocket # 这个要钱，网上找一下教程
  • 安卓：V2rayNG
